Service: Broker

POST services [host, port, name] - Register new service
GET services [] - Fetch all services

Service: Log

POST log/logs [host, port, service, message] - Create new log
GET log/logs [] - Fetch all logs

Service: Auth

POST auth/users/register [username, password, passwordConfirm] - Register user
PUT auth/users/{ID} [password, passwordConfirm, passwordOld] - Update user
DELETE auth/users/{ID} [] - Delete user
GET auth/users/{ID} [] - Fetch user
POST auth/users/login [username, password] - Login user
POST auth/users/logout [] - Logout user
GET auth/auth [method, path] - Fetch user authorization

POST auth/teams [name, creator] - Create team
PUT auth/teams/{ID} [users] - Update team
DELETE auth/teams/{ID} [] - Delete team
GET auth/teams/{ID} [] - Fetch team

POST auth/projects [name] - Create project
PUT auth/projects/{ID} [teams] - Update project
DELETE auth/projects/{ID} [] - Delete project
GET auth/projects/{ID} [] - Fetch project

DELETE core/models/{ID} [] - Delete model
GET core/models/{ID} [] - Fetch model

POST core/elements [type, model, details] - Create element
PUT core/elements/{ID} [details] - Update element
DELETE core/elements/{ID} [] - Delete element
